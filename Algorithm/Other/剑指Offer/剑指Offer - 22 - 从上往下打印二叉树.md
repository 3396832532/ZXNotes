#### 题目

> 从上往下打印出二叉树的每个节点，同层节点从左至右打印。

### 解析

可能是剑指Offer最简单的一道题了。。

直接上二叉树层次遍历代码即可。

代码:

```java
import java.util.*;

public class Solution {

    public ArrayList<Integer> PrintFromTopToBottom(TreeNode root) {
        ArrayList<Integer> res = new ArrayList<>();
        if (root == null)
            return res;
        Queue<TreeNode> queue = new LinkedList<>();
        queue.add(root);
        TreeNode cur = null;
        while (!queue.isEmpty()) {
            cur = queue.poll();
            res.add(cur.val);
            if (cur.left != null)
                queue.add(cur.left);
            if (cur.right != null)
                queue.add(cur.right);
        }
        return res;
    }
}
```
